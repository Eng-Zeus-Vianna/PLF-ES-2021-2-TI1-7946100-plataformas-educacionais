# Plataformas Educacionais

O objetivo do trabalho é a disponibilização de uma plataforma online, na qual o usuário pode cadastrar cursos e aulas contidas dentro deles a fim de organizar o seu progresso. A plataforma tem objetivo de propocionar confiança e organização nos estudos.
